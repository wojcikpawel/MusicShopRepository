## Technologies
Project is created with:
* C# - version 8.0
* Windows Forms - version 2.1.3
* Visual Studio - version 14.0.25422.01 Update 3 2015 Community

## Features
* playing .mp3 files
* virtual music shop
* export music playlists to .pdf file
